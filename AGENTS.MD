# Nota para asistentes de código

**FincAirbnb** es un MVP de estudio: marketplace ficticio de fincas en gallego, con Next.js (App Router) y un backend Express aparte (`FincAirbnb_backend`).

1. Lee el **[README.md](./README.md)** del repo.
2. Contrato y API: `docs/development/API_CONTRACT_ALIGNMENT.md`.
3. Idioma de la UI: **gallego** (`gl` en layout).
4. No commitees `.env` ni `.env.local`; los secretos solo en el servidor o variables del proveedor.

Para detalle de negocio y tono de marca, si aún existe, **`RESUMEN_EJECUTIVO.md`**.
