# SDD Skills Registry (Frontend + Backend)

## Objetivo

Definir skills aprobadas para flujo Spec-Driven Development en ambos proyectos.

## Skill externa validada

- Repo: `https://github.com/shubhamsaboo/awesome-llm-apps`
- Skill: `fullstack-developer`
- Referencia validada: `https://skills.sh/shubhamsaboo/awesome-llm-apps/fullstack-developer`

Comando de ejemplo:

```bash
npx skills add https://github.com/shubhamsaboo/awesome-llm-apps --skill fullstack-developer
```

## Criterios de aceptación para cualquier skill externa

1. Debe exponer `SKILL.md`.
2. Debe estar versionada y accesible por URL estable.
3. Debe pasar revisión interna de seguridad y calidad.
4. Debe documentarse su uso en este registro.

## Skills recomendadas para este proyecto

- `fullstack-developer` (soporte general frontend/backend).
- Skill interna `fincairbnb-sdd-workflow` (a crear) para enforce de spec -> implementación.
- Skill interna `fincairbnb-api-contract-review` (a crear) para validación de OpenAPI y breaking changes.
