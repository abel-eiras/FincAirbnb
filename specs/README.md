# Spec-Driven Development (Frontend)

Este directorio define el flujo SDD para `D:/Webs/FincAirbnb`.

## Estructura

- `_templates/feature-spec.md`
- `_templates/test-cases.md`
- `_templates/traceability.md`
- `_templates/adr.md`
- `features/<feature-slug>/`
- `adr/`

## Flujo obligatorio

1. Crear `spec.md` desde plantilla.
2. Crear `test-cases.md`.
3. Crear `traceability.md`.
4. Revisar/aprobar spec.
5. Implementar código.
6. Actualizar trazabilidad y estado final.
